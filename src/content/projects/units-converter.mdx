## Overview
Units Converter is a user-friendly app designed for effortless conversion between imperial and metric systems. This static website simplifies the process of converting various units across different categories, including distance, weight/mass, volume, temperature, and currency. Whether converting between kilometer and miles, grams and ounces, or Celsius and Fahrenheit, this tool provides quick and accurate results, making it useful for a wide range of applications.

## Tech Stack
- `React`: Library used to develop single page applications.
- `SCSS`: Preprocessor scripting language that is a superset of CSS.
